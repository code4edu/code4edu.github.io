Code 4 Education Hackathon

СГУ им. Н.Г. Чернышевского совместно с компанией EPAM Systems представляют хакатон - марафон программирования среди студентов, который пройдет 19 декабря 2015 года в 12 корпусе университета.

Code 4 Education - это возможность реализовать собственные идеи и разработать решения, которые могли бы найти применение в современной сфере образования. Это могут быть инструменты, настольные или веб приложения, порталы или сайты, для школ, ВУЗов, других образовательных учреждений, преподавателей, учащихся и конечно же самих студентов!

Хакатон Code4Education дает уникальную возможность самостоятельно реализовать то, чего на ваш взгляд не хватает современному процессу образования, получить опыт работы в команде в экстремальных условиях и посоревноваться с соперниками!

Правила

За 8 часов необходимо разработать работающий продукт в рамках представленной темы.

Подготовка

Соберите команду

Найдите единомышленников, придумайте команде название и зарегистрируйтесь

Придумайте идею

Придумайте идею продукта для реализации в рамках темы хакатона

Запланируйте разработку

Выберите любой язык программирования, подготовьте компьютер, установите необходимый инструментарий и среду разработки

Познакомьтесь с Git

Заведите GitHub аккаунт и установите Git себе на компьютер. Ознакомьтесь с системой контроля версий Git

Реализация

Разработка

Пишите код параллельно или применяя парное программирование. Равномерно заносите создаваемый код в предоставленный организаторами репозиторий на GitHub

Консультации

Консультируйтесь по техническим вопросам, в частности по работе с Git, с кураторами, предоставленными организаторами

Разрешается

Пользуйтесь интернетом и открытыми библиотеками, используйте любые другие ресурсы

Запрещается

Не используйте собственные наработки кода и не привлекайте к разработке третьих лиц
